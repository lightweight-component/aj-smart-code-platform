# 数据服务 AJ DataService 简介

<h4>只需写 SQL 业务逻辑（甚至零代码不写），即可快速搭建 CRUD 接口服务</h4>

<p style="text-align: right;font-style: italic;">——最简单的方式：零代码，在页面上配置好参数，自动生成 SQL 并且直接转化成 HTTP API。</p>
